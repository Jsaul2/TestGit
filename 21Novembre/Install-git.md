## Installer GIT

#### Windows

Telecharger sur: `https://git-for-windows.github.io/`

Configurer:

```
git config --global user.name "Votre nom ou pseudo"
git config --global user.email "Votre@email.com"
```

#### Mac et Linux

Telecharger sur: `http://git-scm.com/downloads`

Configurer:

```
git config --global user.name "Votre nom ou pseudo"
git config --global user.email "Votre@email.com"
```
